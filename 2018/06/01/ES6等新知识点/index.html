Object.keys()

用于获取对象自身所有的可枚举的属性值，但不包括原型中的属性，然后返回一个由属性名组成的数组。

Object.getOwnPropertyNames()

返回对象的所有自身属性的属性名（包括不可枚举的属性）组成的数组，但不会获取原型链上的属性。

参考资料：https://www.cnblogs.com/goloving/p/7680572.html

some()与every()数组判断

some()：一真即真，只要其中一个为true 就会返回true的。

every()：一假即假，只要其中一个味false就会返回false的。

filter()过滤数组

给需要过滤的数组调用filter()函数，并在括号中填写滤掉条件，并返回过滤完的数组传给变量。

import { mapGetters } from “vuex”（结构赋值）

定义：讲单独一个或者多个方法引入给文件

优点：调用引入的文件中的方法，把需要调用的方法名暴露给引入的文件中，这样就不用全部方法都引入。

…（扩展运算符）

展开一个对象，将里面的内容都扁平化展现出来。数组直接写函数名当做需要在此页面调用的全局方法，可以直接调用获取state中的值，对象的调取方式为使用箭头函数获取state文件，然后获得state内的对象值

${ 变量 }（Template Strings 模版字符串）

介绍：用来返回字符串，可以直接将变量（或者计算时的变量）放进里面然后转换为字符串输出
